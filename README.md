# Mini Banking System (Java)

## Description
A console-based banking system built using Java and Object-Oriented Programming concepts.

## Features
- User registration and login
- Deposit and withdrawal functionality
- Balance checking
- Menu-driven interface

## Concepts Used
- OOP (Encapsulation, Classes, Objects)
- ArrayList
- Loops and Conditionals

## How to Run
1. Compile:
   javac BankSystem.java

2. Run:
   java BankSystem

## Future Improvements
- File storage (save accounts permanently)
- GUI using Java Swing
